INSTRUCCIONES PARA DESPLEGAR EN RENDER

Crea una cuenta en Render: Ve a render.com y regístrate (usa GitHub para login rápido).

Conecta tu repositorio GitHub:

En el dashboard de Render, haz clic en "New" > "Web Service".
Selecciona "Connect" y elige tu repo alba_unrc de GitHub.
Render detectará automáticamente que es un proyecto Python.
Configura el despliegue:

Nombre del servicio: Pon algo como "my-ml-app".
Entorno: Selecciona "Python 3".
Build Command: Deja vacío (Render instala automáticamente desde requirements.txt).
Start Command: NO cambies esto - Render usa el Procfile automáticamente.
Variables de entorno: No necesitas PORT, Render lo maneja automáticamente.
Plan: Elige "Free" para empezar (tiene límites de 750 horas/mes).
Despliega:

Haz clic en "Create Web Service". Render clonará tu repo, instalará dependencias y arrancará el servidor.
Espera 5-15 minutos. Una vez listo, te dará una URL como https://my-ml-app.onrender.com.
Tu frontend estará en la raíz (/), y el endpoint /clasificar funcionará.

VERIFICACIÓN DEL DESPLIEGUE:

1. Abre la URL en tu navegador: deberías ver tu página HTML con la mascota.
2. Verifica el health check: https://my-ml-app.onrender.com/health debería retornar {"status": "healthy", "model_trained": true}
3. Prueba el endpoint: Usa Postman o curl para enviar un POST a https://my-ml-app.onrender.com/clasificar con JSON {"texto": "me gusta estudiar"}

SI HAY ERRORES:

- Revisa los logs en Render (pestaña "Logs") para ver mensajes de error específicos.
- Los errores comunes son: dependencias faltantes, tiempo de inicio excedido, o problemas de memoria.
- Si el modelo no funciona, podría ser que el entrenamiento tome demasiado tiempo - considera optimizar el código.

NOTAS IMPORTANTES:

- El modelo se entrena cada vez que la app se inicia (normal para este tamaño).
- Si excede el tiempo límite de Render (15 min), considera reducir el tamaño del dataset o usar un modelo pre-entrenado.
- El plan Free de Render duerme la app después de 15 min de inactividad, por lo que la primera carga puede ser lenta.
